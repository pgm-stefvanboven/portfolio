ThinkTap - Game

Wat?

ThinkTap is een spelletje dat is ontwikkeld voor het vak @work2. Het doel is om mensen van buitenaf kennis te laten maken met coderen.

Hoe?

Het spel is gebaseerd op boter, kaas en eieren waar het de bedoeling is om als eerste 3 X'en of O's te hebben. De code is afkomstig van het internet en is door mij en mijn collega aangepast.

Wie?

Dit project is gemaakt door mezelf en mijn collega Lennert Vermoesen.
