CV - Opdracht

Wat?

Voor het vak User Interface Design hebben we een fictieve CV ontworpen om dit vervolgens om te zetten naar een één pagina website. Het doel van deze opdracht was om te leren hoe we een ontwerp konden omzetten naar een functionerende één pagina website.

Hoe?

Het ontwerp van mijn CV is gemaakt in Figma en de één pagina website is gemaakt met HTML en CSS.

Wie?

Dit project is met trots gemaakt door mezelf.
