Afstudeerprojecten - Project

Meer uitleg over dit Project.

Wat?

Voor het vak Programming 1 moesten we een website maken met javascript en daarin de afstudeerprojecten laten zien van vorige studenten. Het doel van deze opdracht was om te leren de lessen van programming 1 toe te passen in deze opdracht. U zult zien dat er van styling weinig sprake is, dit komt omdat het vooral om de functionaliteit gaat.

Hoe?

De website is gemaakt met HTML en CSS maar vooral met javascript. Voor de uitwerking werd het DOM model gebruikt.

Wie?

Dit project is met trots gemaakt door mezelf (Stef Van Boven).
