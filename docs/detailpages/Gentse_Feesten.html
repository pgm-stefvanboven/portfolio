Gentse Feesten 2022 - Fictieve website

Wat?

Voor het vak @work1 maakte ik de website van de Gentse Feesten editie 2022 na. Het doel van deze opdracht was om de lessen van Programmeren 1, Web Design en Computer systems toe te passen op echte opdrachten vanuit het werkveld.

Hoe?

Deze website is gemaakt met HTML, CSS en Javascript. De focus van deze website lag zowel op het visuele als op de functionaliteiten. De front-end is gemaakt met HTML en CSS. De functionaliteiten zijn gemaakt met Javascript.

Wie?

Dit project is met trots gemaakt door mezelf.
